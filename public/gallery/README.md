# Gallery images

Drop your screenshots here as `.jpg` or `.webp` (max ~500 KB each).
Filenames must match the entries in `lib/config.ts` → `gallery`:

- `spawn.jpg`     — spawn area, big tile
- `market.jpg`    — market district
- `forge.jpg`     — public forge
- `skyblock.jpg`  — skyblock world (wide tile)
- `festival.jpg`  — event/festival shot
- `hardcore.jpg`  — hardcore world (tall tile)
- `build1.jpg`    — member build showcase
- `sunset.jpg`    — landscape / mood shot

Captions live in `lib/translations/*.ts` under `gallery.captions`.
Missing files fall back to a styled gradient automatically.
