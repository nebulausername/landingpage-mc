## 🚀 Deploy options

Three paths, depending on your skill level:

1. **One-click Vercel deploy** — pushes to GitHub, click Deploy, done in 60 seconds. Free hosting. No terminal needed.
2. **Local development** — included setup.sh / setup.bat scripts; double-click and you're running.
3. **Static export** — uncomment one line in next.config.mjs, run `npm run build`, upload the `out/` folder anywhere (Hostinger, FTP, Netlify, GitHub Pages).

## ✏️ What you'll customize

Every server detail lives in **one file** (`lib/config.ts`):
- Server name, IP, version
- Discord and store URLs
- World list (Survival, Creative, Hardcore, SkyBlock, Arena — toggle or rename any)
- Rank list with prices and buy URLs
- FAQ count
- Default theme and language
- Optional live status API URL

Text content lives in three translation files (one per language). Colors live in `app/globals.css` (12 light tokens + 12 dark tokens).
