## 💬 First-Reply-Templates

**"How easy is setup?"**
> Setup ist wirklich einfach — 3 Min mit Vercel one-click, oder Doppelklick auf `setup.bat` für lokal. Die README hat einen sehr ausführlichen Schritt-für-Schritt-Guide. Falls du irgendwo stecken bleibst: schreib mir, ich helfe gerne durch.

**"Can I customize the colors?"**
> Yes — all 12 colors are CSS variables at the top of `app/globals.css`. Change values, save, deploy automatically. Same for dark mode (separate set of 12).

**"Does the newsletter need a database?"**
> No — submissions are forwarded to a Discord webhook of your choice. Zero database required. You read signups in your Discord channel as nice embed messages. Set one env var (`CONTACT_WEBHOOK_URL`) in Vercel and you're done. Rate-limiting is built-in.

**"Can I add a 4th language?"**
> Yes — copy `lib/translations/en.ts` to a new file (e.g. `fr.ts`), translate the strings, add `'fr'` to `config.languages` and to the `Lang` type, register it in `lib/translations/index.ts`. The language switcher picks it up automatically. README has the exact steps.

**"Will it work without the Discord widget?"**
> Yes — if you leave `discordWidgetId` as null, the section shows a stylized placeholder so the layout stays balanced. Set the ID later when ready.
